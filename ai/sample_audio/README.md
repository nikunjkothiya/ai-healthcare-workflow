# Sample Audio Files

Place sample WAV audio files here for batch campaign testing.

The worker will use these files to simulate patient conversations.

Files should be:
- WAV format
- 16kHz sample rate
- Mono channel
- 16-bit PCM

You can generate test audio files using:
- Text-to-speech tools
- Recording actual audio
- Converting existing audio files

Example filenames:
- patient1.wav
- patient2.wav
- patient3.wav
